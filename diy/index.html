Mācību materiāls: HTML pamati

Izpēti soļus, lai izveidotu HTML dokumentu ar stilīgu dizainu.

Satura rādītājs

  • 1. solis: HTML pamati
  • 2. solis: Stila pievienošana
  • 3. solis: Slaidrādes veidošana

1. solis: HTML pamati

Šeit tu uzzināsi, kā uzbūvēt HTML struktūru.

  • Izmanto semantiskās tagus, piemēram, <header> un <main>.
  • Pievieno alt tekstu attēliem.
  • Atceries par strukturētu izkārtojumu.

2. solis: Stila pievienošana

CSS piešķir dokumentam dizainu un vizuālu organizāciju.

  • Pievieno ārējo CSS failu vai izmanto iekšējo <style> tagu.
  • Izmanto selektorus, lai stilizētu dažādus elementus.
  • Piemēram, definē teksta krāsas, izmērus un izkārtojumu.
